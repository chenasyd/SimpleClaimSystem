## 1. 基础概念

### 1.1 什么是区块 (Chunk)？

区块是 Minecraft 世界中的 16×16×世界高度 的方块区域。当你站在某个位置执行命令时，系统会识别你当前所在的区块。

在现代 Minecraft（1.17+）中，世界高度不再固定为 256 格。新版世界采用双向延伸设计：

| Minecraft 版本 | 最低高度 | 最高高度 | 可用高度 |
|----------------|----------|----------|----------|
| 1.16 及之前 | 0 | 255 | 256 格 |
| 1.17 - 1.20 | -64 | 319 | 384 格 |
| 1.21+ | -64 | 319 | 384 格 |

SimpleClaimSystem 使用 Spigot/Paper API 的 `world.getMinHeight()` 和 `world.getMaxHeight()` 方法，**自动适配**你服务器运行的 Minecraft 版本世界高度限制，无需手动配置。

### 1.2 领地 vs 保护区域

| 类型 | 说明 | 所有者 |
|------|------|--------|
| **领地 (Claim)** | 玩家创建的领地 | 普通玩家 |
| **保护区域 (Protected Area)** | 管理员创建的区域 | 服务器 (*或管理员) |

### 1.3 领地组成

- **一个领地可以包含多个区块**
- **成员**：被允许进入和使用领地的玩家
- **禁止玩家**：被禁止进入领地的玩家
- **权限设置**：针对不同角色的独立权限

### 1.4 角色类型

| 角色 | 说明 |
|------|------|
| **所有者 (Owner)** | 领地的创建者，拥有完全控制权 |
| **成员 (Member)** | 被邀请加入的玩家，有部分权限 |
| **访客 (Visitor)** | 未被邀请的玩家，受领地权限限制 |
| **自然 (Natural)** | 影响领地的自然事件（爆炸、液体流动等） |

## 2. 领地命令

### 2.1 基础命令

#### 创建领地
```
/claim 或 /claim <半径>
```
- 在空白区域执行，创建新领地
- 可指定半径一次圈多块（如 `/claim 1` 圈 3×3=9 块）
- 可能需要支付金币（取决于服务器配置）

#### 查看领地列表
```
/claim list
```
- 打开 GUI 查看你的所有领地

#### 传送至领地
```
/claim tp <领地名>
```
- 传送到指定领地的出生点

#### 取消认领
```
/unclaim 或 /unclaim <领地名> 或 /unclaim *
```
- 删除你的一块或所有领地
- 删除后该区块变为空白，可被他人认领

---

### 2.2 领地管理命令

#### 重命名领地
```
/claim setname <旧名称> <新名称>
```

#### 设置领地描述
```
/claim setdesc <领地名> <描述文字>
```

#### 设置领地出生点
```
/claim setspawn
```
- 设置玩家传送至该领地时的降落位置

#### 查看领地地图
```
/claim map      - 单次显示地图
/claim automap  - 切换区块时自动显示
```

### 2.3 成员管理命令

#### 添加成员
```

/claim add * <玩家名>              - 添加到你的所有领地
```

#### 移除成员
```
/claim remove <领地名> <玩家名>    - 从指定领地移除

```

#### 查看成员
```
/claim members <领地名>
```
- 打开 GUI 查看领地成员列表

#### 查看禁止列表

### 2.5 驱逐玩家命令

```
/claim kick <领地名> <玩家名>       - 将玩家从指定领地踢出
/claim kick * <玩家名>              - 从所有领地踢出
```

---

### 2.6 领地扩建命令

#### 添加区块
```
/claim addchunk <领地名>
```
- 在**自己领地旁边**的区块执行，将该区块添加到已有领地
- **注意**：如果你在**自己领地旁边**执行 `/claim`，会创建**独立的新领地**而非扩建
- 扩建费用通常比创建新领地便宜

#### 删除区块
```
/claim delchunk <领地名> <世界>;<X>;<Z>
```
- 从领地中移除特定区块
- 示例：`/claim delchunk myhome world;0;1`

#### 查看领地区块
```
/claim chunks <领地名>
```
- 打开 GUI 查看领地包含的所有区块

---

### 2.7 合并领地命令

#### 合并两块领地
```
/claim merge <领地名1> <领地名2>
```
- 将两块相邻的独立领地合并为一块
- 两块领地必须是相邻的

## 5. 经济系统

### 5.1 费用计算

根据服务器配置，认领和添加区块可能需要支付金币：

| 操作 | 说明 |
|------|------|
| **创建领地** | 每次执行 `/claim` 创建新领地时的费用 |
| **添加区块** | 使用 `/claim addchunk` 扩建领地时的费用 |

### 5.2 递增收费

如果服务器开启了递增收费：
- 第 1 块领地 = 基础价格
- 第 2 块领地 = 基础价格 × 倍数
- 第 3 块领地 = 基础价格 × 倍数²
- 以此类推...

### 5.3 VIP 优惠

不同玩家组可能有不同的费率折扣，详情咨询管理员。

### 5.4 购买领地

1. 找到待售领地（会有特殊显示）
2. 站在该领地上
3. 执行 `/claim buy`
4. 金币会从你的账户扣除，领地所有权转移给你

## 8. 常见问题

### Q1: 为什么我不能在某处创建领地？
可能原因：
- 该区域已有他人领地
- 该区域是保护区域
- 你已达到最大领地数量限制
- 你的金币余额不足
- 当前世界禁用了领地功能

### Q2: 领地旁边的区块被他人认领了怎么办？
如果需要，你可以：
1. 与该玩家协商让他释放区块
2. 使用 `/claim addchunk <他的领地名>` 将区块添加到他领地（需要他有足够权限）
3. 寻找其他位置创建领地

### Q3: 如何将两个独立的领地合并？
使用 `/claim merge <领地名1> <领地名2>` 命令
- 两块领地必须相邻
- 合并后成为一块领地

### Q4: 领地可以转让给其他人吗？
可以，使用 `/claim owner <领地名> <新所有者>` 命令

### Q5: 如何开启自动认领？
执行 `/claim autoclaim` 开启自动认领模式

### Q6: 领地支持经济系统吗？
支持！服务器需要安装 Vault 和经济插件。你可以使用 `/claim sell` 出售领地，其他玩家可以用 `/claim buy` 购买。
